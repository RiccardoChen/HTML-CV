# HTML-CV :star_struck:
A personal CV project built using pure HTML.

## Features
* 100% Semantic HTML5 with some Inline and Internal CSS
* No external CSS or JavaScript used
* Clean and accessible structure

## Keypoints
* Semantic HTML
* SEO Meta Tags
* Open Graph (OG) Tags
* Favicon
